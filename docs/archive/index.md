## Archive Organization

All archived documents use date-prefixed naming: `YYYY-MM-DD-document-name.md`

This makes it easy to see:

- When documentation was created
- Timeline of project evolution
- Historical context for decisions

Documents are organized by category to make browsing easier while maintaining chronological information in the filename.
