# nano commands

```sql
# File handling
Ctrl+S         save current file
Ctrl+O         save as (offer to write)
Ctrl+R         insert file into current
Ctrl+X         close / exit

# Editing
Ctrl+K         cut line to cutbuffer
Alt+6          copy line to cutbuffer
Ctrl+U         paste cutbuffer
Ctrl+]         complete current word
Alt+3          comment / uncomment line
Alt+U / Alt+E  undo / redo

# Search and replace
Ctrl+W         search forward
Ctrl+Q         search backward
Alt+W / Alt+Q  next match forward / backward
Alt+R          start replacing session

# Deletion
Ctrl+H         delete char before cursor
Ctrl+D         delete char under cursor
Alt+Bsp        delete word left
Ctrl+Del       delete word right
Alt+Del        delete current line

# Navigation
Ctrl+A / Ctrl+E   start / end of line
Ctrl+P / Ctrl+N   line up / down
Ctrl+Y / Ctrl+V   page up / down
Alt+\ / Alt+/     top / bottom of file
Alt+G              go to line number

# Selection and indent
Alt+A          toggle mark (selection start)
Tab            indent marked region
Shift+Tab      unindent marked region

# Macros
Alt+:          start / stop recording macro
Alt+;          replay macro
```
