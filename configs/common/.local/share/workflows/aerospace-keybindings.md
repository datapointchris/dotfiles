# AeroSpace keybindings

```text
# Main Mode
alt + h/j/k/l                     focus window (directional)
alt + shift + h/j/k/l             move window (directional)
alt + a/d/e/m/s/x/z               go to workspace (direct)
alt + shift + a/d/e/m/s/x/z       move window to workspace
alt + f                            toggle floating / tiling
alt + shift + f                    toggle fullscreen
alt + '                            toggle tiles horizontal / vertical
alt + \                            toggle accordion horizontal / vertical
alt + tab                          switch to previous workspace
alt + enter                        open new Ghostty terminal
alt + r                            enter Resize Mode
alt + shift + r                    reload config
alt + shift + ;                    enter Service Mode

# Join / Group
Ctrl+Shift+Alt + h/j/k/l          join with window (directional)
Ctrl+Shift+Alt + g                 flatten workspace tree

# Resize Mode (alt + r to enter)
h/j/k/l                           resize window (directional)
esc or enter                       return to Main Mode

# Service Mode (alt + shift + ;)
esc                                reload config & return to main
r                                  reset / flatten layout
f                                  toggle floating / tiling
```
