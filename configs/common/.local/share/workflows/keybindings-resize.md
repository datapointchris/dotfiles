# keybindings — resize across AeroSpace, Tmux, Neovim

```text
# AeroSpace
alt + r                           enter Resize Mode
  h/j/k/l                         resize window (directional)
  esc or enter                     exit Resize Mode

# Tmux
Ctrl + Alt + h/j/k/l              resize panes (5 units)

# Neovim
<leader>r + h/j/k/l               resize splits (10 units)
<leader>rm                         maximize / minimize current split
```

Ctrl+Alt+hjkl resizes Tmux panes even from within Neovim.
